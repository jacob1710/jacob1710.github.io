StuBank

Stubank was a project created as part of a team software engineering module in which as part of a group of 5 we created a student banking web application, with machine learning implemented analysing transactions to prevent fraudulent transactions from going through. The styling of the website used the Halfmoon css framework, and we created our own custom stylesheet to modify elements to adjust it to our designs.

Machine Learning

The machine learning was one of the main parts of the system, and was used as to prevent any transactions from occurring that were fraudulent. It was our job to decide what fraudulent actually meant and how we would go about preventing those transactions. We decided to use several measures to try and distinguish normal transactions from abnormal transactions:

  • Comparing the location of the purchase with the users previous transactions to try and determine their usual spending habits and work out if it was in an unusual location for this user.
  • Comparing the time and value of the the purchase compared to their usual amounts by creating clusters of their previous times and amounts and analysing whether the new purchase would be part of any of these.
  • Using all these methods to influence a value on how abnormal this transaction is, which ultimately determines whether we block it or not.
