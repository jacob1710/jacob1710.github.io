StuBank is a Student Banking system with integrated machine learning, and a personalised stories feature showing a users recent transactions and transactional trends. It was built using Python for the backend with the Flask Framework for the front end, with parts involving JavaScript, and the database being a MySQL database stored on Google Cloud.

NoteyIO

NoteyIO is a full stack app platform that was built using Flutter for the front end, a Node.js server using the Express framework for the backend, and storage in MongoDB. DevOps practices were used throughout the development including a CI/CD pipeline, server monitoring, and the whole project being ran in Docker containers.
