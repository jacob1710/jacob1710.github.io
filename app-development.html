NoteyIO

NoteyIO is a full stack app platform that was built using Flutter for the front end, a Node.js server using the Express framework for the backend, and storage in MongoDB. DevOps practices were used throughout the development including a CI/CD pipeline, server monitoring, and the whole project being built into Docker containers.
