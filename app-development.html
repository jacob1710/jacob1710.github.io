NoteyIO

NoteyIO is a full stack app platform that was built using Flutter for the front end, a Node.js server using the Express framework for the backend, and storage in MongoDB. DevOps practices were used throughout the development including a CI/CD pipeline, server monitoring, and the whole project being built into Docker containers. Wazos

Wazos was a full stack application built using Flutter for the front end, Springboot Java for the backend and data models, and connected to a MySQL database stored in AWS. The application was a storefront app that was built upon the stores existing Shopify platform.

RePOS

RePOS was a point of sale platform built using Flutter for the front end, Springboot Java for the backend and data models, and connected to MySQL database stored in AWS. The app was designed to be used by shops who would scan products using the barcode scanner, or find them through the search system. They would then place an order and state whether the customer was paying through cash or through a payment link. This payment link would display as a QR code for the customer to scan and they can then pay through the web portal. The app can display all the previous orders, and can bring up the payment link
